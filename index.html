STEPS Decision Aid Tool

This tool supports decision makers in planning and scaling Field Epidemiology Training Programs (FETP) in India by predicting programme uptake, performing detailed cost–benefit analysis, and evaluating willingness‐to‐pay (WTP) for key features.

STEPS predicts participant uptake, computes cost–benefit outcomes, and evaluates the extra value (WTP) that programme features provide.

  • Instructions: Step-by-step guidance.
  • Attributes: Review programme features with icons and tooltips.
  • Inputs: Configure your scenario using input cards.
  • WTP Results: View extra value (in USD) for each feature relative to a baseline.
  • Predicted Program Uptake: See the estimated participation uptake with recommendations.
  • Costs & Benefits: Detailed analysis with educational summaries.
  • Scenarios: Save, compare, and export your configurations.

How to Use This Tool

  1. Review the Attributes tab to learn about each programme feature. Hover over icons for details.
  2. In the Inputs tab, select your desired levels using input cards (each displays its reference) and adjust the sliders.
  3. Click "Calculate & View Results" to open a popup modal with a summary of predicted uptake and recommendations.
  4. Examine the WTP Results tab for a bar chart comparing each attribute level (except the reference) to the baseline.
  5. Review the Predicted Program Uptake tab for a full uptake chart with detailed recommendations.
  6. Check the Costs & Benefits tab for a detailed breakdown of cost components and benefit calculations with an educational explanation.
  7. Save your scenario in the Scenarios tab to compare configurations or export a PDF report.

Costs & Benefits Analysis

Total costs are calculated as (Cohort Size × Training Cost) where costs include fixed components (local press ads, printing, postage, administrative personnel, trainer fees, on-costs, facilitator salaries) and variable components (material costs, venue hire, session time, travel). Benefits are estimated from QALY gains (assumed 0.05 per participant) multiplied by $50,000 per QALY.
