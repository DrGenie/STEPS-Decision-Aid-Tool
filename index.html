STEPS Decision Aid Tool

This tool helps decision makers in India plan and scale Field Epidemiology Training Programs (FETP) by predicting programme uptake, performing cost–benefit analyses, and evaluating willingness‐to‐pay for key attributes.

Introduction

  • Instructions: Step-by-step usage guide.
  • Attributes: Four discrete (with references) plus two continuous (Cohort Size; Cost mapped \$60–\$1500). Tooltips in smaller font, color #2e4053 background.
  • WTP Results: Extra value for each non-reference discrete level plus a +1 increment for the continuous attributes.
  • Costs & Benefits: QALY scenario (Low=0.01, Mod=0.05, High=0.08) for India, cost breakdown cards.

How to Use This Tool

  1. Review each Attribute for discrete references or continuous ranges. Hover over icons for details (0.75em tooltips).
  2. In Inputs,
    • Select non-reference radio buttons for discrete attributes (Training, Delivery, Accreditation, Location). Reference levels are not selectable.
    • Pick a Cohort Size slider value (range 500–2000). The selected value is displayed in real-time.
    • Pick a Cost slider value from 0–250, linearly mapped to \$60–\$1500. E.g., slider=125 => about \$780.
  3. Click “Calculate & View Results” to see predicted uptake. The Predicted Program Uptake tab displays a doughnut chart with the new estimate.
  4. Check WTP Results for a bar chart of each non-ref discrete level plus +1 increments in the continuous attributes, with ±10% error bars.
  5. Under Costs & Benefits, pick a QALY scenario (Low=0.01, Mod=0.05, High=0.08). A table and bar chart update showing total cost, net benefit, QALYs, and monetized benefits. Educational text explains cost components.
  6. Scenarios can be saved or reloaded. You can export them to PDF for easy reference in policy discussions.
