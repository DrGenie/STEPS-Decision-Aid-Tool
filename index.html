STEPS Decision Aid Tool

This tool helps decision makers plan and scale Field Epidemiology Training Programs (FETP) in India by estimating programme uptake, costs, benefits, and willingness‐to‐pay for key attributes.

STEPS (Scalable Training Estimation and Planning System) is designed to support decision makers in optimizing Field Epidemiology Training Programs (FETP) across India. By estimating how many participants will enroll, calculating detailed cost–benefit outcomes, and assessing the monetary value of various programme features, STEPS provides clear, actionable insights.

  • Instructions – Follow simple steps to use the tool.
  • Attributes – Review key programme features with icons and tooltips.
  • Inputs – Set up your scenario using user-friendly controls.
  • WTP Results – View willingness-to-pay estimates for each attribute level.
  • Predicted Program Uptake – See the estimated participation rate.
  • Costs & Benefits – Examine a detailed, educational cost–benefit analysis.
  • Scenarios – Save and compare different configurations, then export a report.

How to Use This Tool

  1. Review the Attributes tab to understand each programme feature. Hover over icons for more information.
  2. In the Inputs tab, set your scenario by selecting levels and adjusting sliders.
  3. Click "Predict Program Uptake" to update the charts and analysis sections.
  4. Visit WTP Results to see the monetary valuation for each attribute level.
  5. Review Predicted Program Uptake for an overview of estimated participation.
  6. Check Costs & Benefits for a detailed explanation of how total costs, benefits, and net benefits are computed.
  7. Save and compare scenarios in the Scenarios tab, or export them to a PDF report.
