STEPS Decision Aid Tool

This tool helps decision makers in India plan and scale Field Epidemiology Training Programs (FETP) by predicting programme uptake, performing cost–benefit analyses, and evaluating willingness‐to‐pay for six attributes in USD.

Introduction

  • Instructions: Step-by-step usage details.
  • Attributes: Four discrete (with references) plus two continuous attributes for FETP scaling.
  • WTP Results: Extra value for each non-ref discrete level plus +1 increments in continuous attributes. Code fixed for stable tabs.
  • Costs & Benefits: QALY scenario (Low=0.01, Mod=0.05, High=0.08), plus cost breakdown cards for context.

How to Use This Tool

  1. Check the Attributes tab for reference levels or continuous ranges. Hover over icons for 0.75em tooltips.
  2. In Inputs:
    • Select non-reference radio buttons for Training, Delivery, Accreditation, and Location.
    • Adjust Cohort Size (500–2000) and Cost (0–250 => \$60–\$1500).
  3. Click "Calculate & View Results" to see predicted uptake. The Predicted Program Uptake tab displays a doughnut chart.
  4. WTP Results shows each non-ref discrete level plus +1 increments in continuous attributes. Slopes: cost=-0.07, cohort=+0.08.
  5. Costs & Benefits updates after each scenario calculation or QALY selection. Illustrative cost cards are shown.
  6. Scenarios can be saved and exported to PDF, including predicted uptake and net benefit.
